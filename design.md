# Rus Cal AI - Дизайн мобильного приложения

## Обзор приложения
Приложение для подсчёта калорий с использованием ИИ, позволяющее пользователям фотографировать еду и получать мгновенный анализ калорийности и питательной ценности. Поддерживает русский и английский языки, светлый и тёмный режимы.

## Список экранов

1. **Главный экран (Home)** - Основной интерфейс для добавления еды
2. **Экран камеры (Camera)** - Снимок еды для анализа
3. **Результат анализа (Analysis Result)** - Показ калорий и питательных веществ
4. **Дневник (Diary)** - История приёмов пищи за день
5. **Статистика (Statistics)** - Графики и прогресс
6. **Профиль (Profile)** - Данные пользователя и настройки
7. **Настройки (Settings)** - Язык, тема, цели

## Основной контент и функциональность

### Главный экран (Home)
- **Заголовок**: "Сегодня" с датой
- **Калорийность дня**: Большой прогресс-бар (съедено / цель)
- **Кнопка камеры**: Центральная кнопка "Сфотографировать еду"
- **Последние приёмы пищи**: Список последних добавленных блюд (завтрак, обед, ужин)
- **Быстрые действия**: Кнопки для ввода еды вручную или сканирования штрих-кода

### Экран камеры (Camera)
- **Полноэкранный вид камеры**: С направляющей рамкой для еды
- **Кнопка захвата**: Большая круглая кнопка снизу
- **Кнопка отмены**: Для возврата на главный экран
- **Галерея**: Быстрый доступ к последним фото

### Результат анализа (Analysis Result)
- **Фото еды**: Загруженное изображение
- **Название блюда**: Определённое ИИ
- **Основные калории**: Большой шрифт с числом
- **Макронутриенты**: Белки, жиры, углеводы (в граммах)
- **Размер порции**: Слайдер для регулировки
- **Кнопка подтверждения**: Добавить в дневник

### Дневник (Diary)
- **Дата**: Выбор дня
- **Приёмы пищи**: Завтрак, обед, ужин, перекусы (с суммой калорий)
- **Список блюд**: Под каждым приёмом пищи
- **Удаление**: Свайп влево для удаления блюда
- **Итого за день**: Калории, белки, жиры, углеводы

### Статистика (Statistics)
- **Граница калорий**: Линейный график за неделю
- **Макронутриенты**: Круговая диаграмма
- **Прогресс веса**: Если включено отслеживание
- **Тренды**: Недельные/месячные статистики

### Профиль (Profile)
- **Аватар и имя**: Редактируемые поля
- **Цель калорий**: Текущая цель (по умолчанию 2000)
- **Вес и рост**: Для расчётов
- **Активность**: Уровень физической активности
- **История**: Возможность просмотра прошлых дней

### Настройки (Settings)
- **Язык**: Переключатель русский/английский
- **Тема**: Светлый/тёмный режим (или автоматический)
- **Уведомления**: Включить/отключить напоминания
- **Цель калорий**: Редактируемое значение
- **О приложении**: Версия и информация

## Основные пользовательские потоки

### Поток 1: Добавление еды через камеру
1. Пользователь нажимает на главном экране кнопку "Сфотографировать"
2. Открывается экран камеры
3. Пользователь делает снимок еды
4. ИИ анализирует изображение (загрузка)
5. Показывается результат анализа с калориями
6. Пользователь может отрегулировать размер порции
7. Нажимает "Добавить" → блюдо добавляется в дневник
8. Возврат на главный экран

### Поток 2: Просмотр дневника
1. Пользователь переходит на вкладку "Дневник"
2. Видит список приёмов пищи за текущий день
3. Может свайпнуть влево для удаления блюда
4. Может выбрать другую дату для просмотра
5. Видит итоговые калории и макронутриенты

### Поток 3: Просмотр статистики
1. Пользователь переходит на вкладку "Статистика"
2. Видит график потребления калорий за неделю
3. Видит распределение макронутриентов
4. Может переключаться между неделей и месяцем

### Поток 4: Изменение языка и темы
1. Пользователь переходит на вкладку "Профиль" → "Настройки"
2. Выбирает язык (русский/английский)
3. Выбирает тему (светлая/тёмная/автоматическая)
4. Изменения применяются мгновенно

## Типография
- **Заголовки**: SF Pro Display (iOS) / Roboto (Android), 28px, bold
- **Подзаголовки**: SF Pro Display / Roboto, 18px, semibold
- **Основной текст**: SF Pro Text / Roboto, 16px, regular
- **Вторичный текст**: SF Pro Text / Roboto, 14px, regular
- **Мелкий текст**: SF Pro Text / Roboto, 12px, regular

## Компоненты UI
- **Кнопки**: Скруглённые углы (12px), с эффектом нажатия (масштаб 0.97)
- **Карточки**: Скруглённые углы (16px), тень в светлом режиме
- **Прогресс-бар**: Скруглённые углы, анимированное заполнение
- **Табы**: Нижняя навигация с 4-5 вкладками
- **Модальные окна**: Полноэкранные или снизу (bottom sheet)
