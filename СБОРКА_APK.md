# 📱 Инструкция по сборке APK — Калория ИИ

## Что исправлено в коде

| Файл | Что исправлено |
|------|---------------|
| `lib/google-vision.ts` | Заменён Google Vision на **Claude AI** (работает без доп. ключей) |
| `app/(tabs)/diary.tsx` | Подключены реальные данные из хранилища |
| `app/(tabs)/statistics.tsx` | Реальные графики за 7 дней |
| `components/ui/icon-symbol.tsx` | Добавлены все отсутствующие иконки (trash, photo, info и др.) |
| `app.config.ts` | Имя приложения, разрешения камеры, слаг |
| `eas.json` | **Новый файл** — конфиг для сборки APK |
| `.env.example` | Шаблон переменных окружения |

---

## Шаг 1 — Установить зависимости

```bash
# Установить Node.js (если нет): https://nodejs.org/
# Установить pnpm
npm install -g pnpm

# В папке проекта:
pnpm install
```

---

## Шаг 2 — Добавить API ключ

1. Зайди на **https://console.anthropic.com/**
2. Создай аккаунт (бесплатно) → API Keys → Create Key
3. Скопируй ключ (начинается с `sk-ant-...`)
4. Открой файл `.env` в папке проекта:

```
EXPO_PUBLIC_ANTHROPIC_API_KEY=sk-ant-ТВОЙ_КЛЮЧ_СЮДА
```

> ⚠️ Без этого ключа ИИ-анализ не будет работать — приложение запустится, но при анализе фото вернёт заглушку.

---

## Шаг 3 — Зарегистрироваться в Expo (бесплатно)

```bash
# Установить EAS CLI
npm install -g eas-cli

# Войти в аккаунт Expo (или создать на https://expo.dev/)
eas login
```

---

## Шаг 4 — Инициализировать проект

```bash
# В папке проекта:
eas init
```

При вопросе "Would you like to automatically create an EAS project?" — ответь **Y**.

---

## Шаг 5 — Собрать APK

```bash
eas build -p android --profile preview
```

Что произойдёт:
- EAS загрузит код на свои серверы (~1-2 минуты)
- Серверы соберут APK (~10-20 минут)
- Ты получишь ссылку для скачивания APK

**Ссылка придёт в терминал и на email**, например:
```
✅ Build finished.
🍺 Your build is ready: https://expo.dev/artifacts/eas/xxxx.apk
```

---

## Шаг 6 — Установить APK на телефон

1. Скачай APK по ссылке из шага 5
2. Перенеси на Android телефон (через кабель или облако)
3. На телефоне: Настройки → Безопасность → Разрешить установку из неизвестных источников
4. Открой APK и установи

---

## Тест без сборки (запуск на телефоне через Expo Go)

Если хочешь сначала проверить без сборки APK:

```bash
# Установить Expo Go на телефон (из Play Market)
# Запустить проект:
pnpm dev:metro

# Сканировать QR-код в Expo Go
```

> ⚠️ В Expo Go камера и некоторые нативные функции могут работать иначе.

## Возможные проблемы

### "Error: No credentials found"
```bash
eas credentials
# Выбери Android → Generate new keystore
```

### "Package name already exists"
В `app.config.ts` измени `rawBundleId`:
```ts
const rawBundleId = "com.yourname.calolifeai"; // уникальное имя
```

### API ключ не работает
- Проверь что ключ начинается с `sk-ant-`
- Убедись что `.env` файл в корне проекта
- Перезапусти сборку
